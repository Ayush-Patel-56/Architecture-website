# StudioSAHAJ | Architectural Portfolio

A premium, high-contrast architectural portfolio built with Next.js, Framer Motion, and Tailwind CSS. Modern, minimalist, and editorial.

## 🏛️ Project Identity
**StudioSAHAJ** is an interdisciplinary architectural design studio (LLP) rooted in **Ahmedabad (India)** with a global presence across:
- 🇺🇸 Boston, USA
- 🇸🇬 Singapore
- 🇨🇦 Toronto, Canada
- 🇨🇭 Zurich, Switzerland
- 🇨🇳 Shanghai, China

## 🚀 Key Features
- **Cinematic Experience**: Immersive, full-screen hero sections and smooth Framer Motion transitions.
- **Editorial Navigation**: Responsive, theme-aware hamburger menu with smart color inversions.
- **Global Studio Directory**: Integrated studio database with regional contact points.
- **Project Showcase**: Filterable portfolio grid for Cultural, Residential, and Institutional works.
- **Firm Philosophy**: Integrated site sections for "Architecture that feels natural, lives lightly, and belongs effortlessly."

## 💻 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## 🛠️ Development
```bash
npm install
npm run dev
```
